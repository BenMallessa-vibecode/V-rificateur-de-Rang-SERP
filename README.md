# 🕷️ SERP Rank Checker

A **simple, free** SERP checker—no ads, no CAPTCHA, just results.

## 🚀 About This Project
**SERP Rank Checker** is a lightweight, no-nonsense tool for tracking **Google search rankings** using the [Serper.dev](https://serper.dev/) API. 

🔹 **No ads. No distractions. No annoying CAPTCHA.**  
🔹 **Fast and reliable SERP data retrieval.**  
🔹 **Free up to 2,500 queries/month.**  
🔹 **Download results as CSV for further analysis.**  

## 🔑 Get Your API Key
This app uses [Serper.dev](https://serper.dev/), a powerful Google Search API that provides **2,500 free queries per month** (no credit card required).

1. Go to **[Serper.dev](https://serper.dev/)**
2. Sign up for free
3. Copy your **API key**
4. Paste it into the sidebar

## 🎯 Features
✅ **Real-time SERP position tracking**  
✅ **Search by keyword, country, and language**  
✅ **View Top 10 results with title, URL, and snippet**  
✅ **CSV download support**  
✅ **Simple, distraction-free interface**
